# algebra

Flashcards for Algebra — for spaced repetition review via [flashcards](https://github.com/thomasrribeiro/flashcards).

## Coverage

| File | Topic |
|------|-------|
| 01_real_numbers | Number systems, ordering, absolute value, number line |
| 02_exponents_radicals | Integer/rational exponents, radicals, simplification |
| 03_polynomials | Polynomial operations, degree, standard form |
| 04_factoring | GCF, grouping, trinomials, difference of squares, sum/diff of cubes |
| 05_linear_equations | Solving linear equations, literal equations, word problems |
| 06_systems_of_equations | Substitution, elimination, graphical solutions, applications |
| 07_quadratic_equations | Factoring, square root, completing the square, quadratic formula, discriminant |
| 08_functions_graphs | Domain/range, function notation, transformations, composition, inverses |
| 09_rational_expressions | Simplifying, multiplying, dividing, adding, LCD, extraneous solutions |
| 10_radical_equations_complex_numbers | Radical equations, imaginary unit, complex arithmetic |
| 11_inequalities | Linear, compound, absolute value, quadratic, rational, sign charts |
| 12_exponential_logarithmic | Exponential/log functions, properties, equations, compound interest |
| 13_sequences_series | Arithmetic/geometric sequences, series, sigma notation, binomial theorem |
